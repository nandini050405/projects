About This Project

This Insurance Charges Predictor is a web-based application powered by machine learning. It estimates the cost of insurance based on user-provided information such as:

  • Age
  • Gender
  • Body Mass Index (BMI)
  • Number of children
  • Smoking status
  • Region

The model was trained using real-world data to provide accurate and reliable predictions, helping users understand how their profile might impact insurance premiums.
